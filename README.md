# Elegoo Robot Car ECU — Embedded Control System

Dual-ECU embedded firmware for Elegoo Smart Robot Car V4.0.
Built with professional embedded engineering practices.

## Architecture
- Low-Level ECU: Arduino UNO (C++, HAL, PID, I2C, PWM)
- High-Level ECU: ESP32-S3 (FreeRTOS, UART Protocol, WiFi Telemetry)

## Tech Stack
PlatformIO | VS Code | Linux | Unity Tests | GitHub Actions CI
